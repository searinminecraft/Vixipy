{{ _("About Vixipy") }}

{{ _("Vixipy is a free and open source, privacy respecting pixiv frontend, allowing you to enjoy pixiv content with a peace of mind in terms of privacy, speed, and reliability.") }}

{{ _("Why Vixipy?") }}

{{ _("The problem with using pixiv is that it uses so much unnecessary bandwidth, has trackers (Google Tag Manager), is proprietary (and contains proprietary components like ReCAPTCHA), possibly privacy invasive, and is also behind Cloudflare, which you already know has many, many problems.") }}

{{ _("With Vixipy, you can enjoy using most pixiv features with a frontend you can trust. Your browser will never connect to any external server except if you specify an external image proxy server.") }}

{{ _("Also on the plus side, you don't need an account for viewing full artwork resolution, unlike pixiv.") }}

{{ _("How does it compare to other pixiv frontends (PixivFE, etc.)?") }}

{{ _("Vixipy is, and is always, never a competitor of any pixiv frontend. Sure, Vixipy does take inspiration from other frontends, but only for the good, and with proper credit or attribution if needed.") }}

{{ _("Performance-wise, Vixipy is sometimes faster or on par with other pixiv frontends.") }}

{{ _("What makes Vixipy unique?") }}

{{ _("Vixipy not only a frontend that allows you to explore pixiv, but it also aims to be a useful and functional pixiv client similar to PixEz (an Android/iOS pixiv client written in Flutter), that allows you to do things like edit your profile, view notifications, express your opinions and feelings by sending comments and stickers, and more to come.") }}
