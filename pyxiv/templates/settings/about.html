{{ _("Why Vixipy?") }}

{{ _("The problem with using pixiv is that it uses so much unnecessary bandwidth, has trackers (Google Tag Manager), is proprietary (and contains proprietary components like ReCAPTCHA), possibly privacy invasive, and is also behind Cloudflare, which you already know has many, many problems.") }}

{{ _("With Vixipy, you can enjoy using most pixiv features with a frontend you can trust. Your browser will never connect to any external server except if you specify an external image proxy server. The only things that Vixipy will log are the pixiv API requests, your user agent, and image proxy requests (unless you are using an external image proxy server, which you will have to trust it's privacy practices).") }}

{{ _("Also on the plus side, you don't need an account for viewing full artwork resolution, unlike pixiv.") }}
