# Vixipy

</div>

Vixipy [vick-see-pie] 是一个自由开源的 pixiv 前端。它让您能够使用 pixiv 并欣赏作品，无需担心窥探的目光。

## 安装指南
安装 Vixipy 只需遵循这些简单的[说明](./doc/INSTALL.md)
